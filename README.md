# OpenHMILab C-more

OpenHMILab C-more is an open-source resource library for the AutomationDirect C-more CM5 Series Touch Screen HMIs.

This project focuses exclusively on the C-more Programming Software (CM5 Series) version 9 and later.

## Included

- CM5 starter projects
- Reusable HMI screens
- Industrial themes
- Navigation systems
- Reusable objects
- Faceplates
- Alarm screens
- Trend screens
- Dashboard layouts
- Best practices
- Documentation

## Supported Software

- C-more Programming Software (CM5 Series)
- Version 9+

## Supported Hardware

- C-more CM5 Series Touch Screen HMIs
